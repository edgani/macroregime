# Start War Room OS v2.9

For hosted deployment, follow `DEPLOY_NOW.md` and use `app.py` as the entrypoint.

For local Windows use:

```text
RESET_RUNTIME.bat
RUN_STABLE_WARROOM.bat
```

The header must show `v2.9 DECISION-FIRST`.

Read action labels literally:

- `BUILD LONG/SHORT`: aligned setup, conditional on the shown trigger and stop;
- `WATCH LONG/SHORT`: directional watch only;
- `REDUCE / AVOID`: defensive, not a short order;
- `NO TRADE`: no permitted action now.

Alpha Center is the structural extreme-upside radar. Market tabs are timing/execution workspaces.
They are intentionally separate.
